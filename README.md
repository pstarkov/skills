# Skills Repository

Collection of reusable skills and API wrappers for local automation and assistant workflows.

## Structure

- `trello/` - Trello REST API skill, scripts, and endpoint reference.
- `utils/` - utility and helper skills.
- `integrations/` - third-party integration skills.

## Repository Conventions

1. Add each skill in its own directory.
2. Keep usage instructions close to the skill, typically in `SKILL.md`.
3. Prefer checked-in docs and scripts over generated one-off notes.
4. Keep secrets, tokens, and local machine configuration out of the repository.

## Usage

Each skill directory contains its own documentation, entry points, and supporting resources.
